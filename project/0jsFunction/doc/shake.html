Shake

摇一摇

Options

threshold: 1500,//optional, 摇的力度
waiting: 1500,//optional, 摇动后延迟多少时间再执行，一般播放音乐需要1.5秒
timeout: 2000,//optional, 几秒内只能摇一次
music: 'images/shake.mp3',// 摇动时的咔咔声
shaked: function() {
		//do something
	}

Javascript

$(window).shake({
	shaked: function() {
		//do something
	}
});
